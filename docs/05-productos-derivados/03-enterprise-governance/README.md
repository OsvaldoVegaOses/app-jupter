# Enterprise Governance (SSO + Audit + Compliance)

**Producto derivado:** sí (paquete enterprise).  
**Motivación:** `docs/04-arquitectura/valor_negocio.md` marca SSO/OAuth, audit logging y compliance como requisitos enterprise.

---

## Qué es
Un paquete para convertir APP_Jupter (o una variante SaaS) en una oferta enterprise: identidad, auditoría, retención y controles operativos.

## Entregables
- Requisitos de seguridad (SSO/OAuth, roles, límites)
- Especificación de audit logging (qué eventos, campos, retención)
- Políticas (backup/retención, export, borrado)
- Checklist de compliance (GDPR/SOC2) a nivel técnico

## Qué lo hace producto separado
- Se vende como add-on enterprise / on-prem
- Tiene roadmap y validación distinta (auditorías, requisitos legales)
