# Interoperabilidad CAQDAS (Exports)

**Producto derivado:** sí (línea centrada en integración y formatos).  
**Motivación:** en `docs/04-arquitectura/valor_negocio.md` se menciona exportación a herramientas/formato estándar como brecha enterprise.

---

## Qué es
Una línea de producto que convierte resultados de análisis (códigos, citas-texto, memos, categorías/relaciones) en **formatos compatibles** con herramientas CAQDAS y flujos académicos.

## Targets (formatos/herramientas)
- REFI-QDA (intercambio estándar)
- MAXQDA
- NVivo (pendiente)
- ATLAS.ti (pendiente)

## Entregables
- Especificaciones de mapeo (qué entidad va a qué campo)
- Exporters (por formato) y validadores
- Ejemplos de paquetes exportados

## Qué lo hace producto separado
- Mantener compatibilidad con cambios de versiones de herramientas
- QA de import/export y pruebas con proyectos de referencia
- Documentación para usuarios finales (investigadores)
