# App Bibliográfica: Marco Teórico desde Literatura Científica

**Producto derivado:** sí (app nueva, distinta de la app base de entrevistas).  
**Documento base:** `matriz_brechas_marco_teorico_literatura.md`.

---

## Qué es
Una aplicación orientada a **gestionar literatura científica** (papers/libros/artículos) y construir un **marco teórico** con:
- referencias bibliográficas formales (DOI/ISBN/URL)
- trazabilidad (obra → sección/página → fragmento → afirmación)
- export (BibTeX/RIS/CSL-JSON)

## Qué NO es
- No es el flujo actual de APP_Jupter para entrevistas (Grounded Theory sobre corpus propio).

## Estado
- En diseño: brechas y criterios definidos en la matriz.
