# Guia de modos epistemicos

Este documento resume como usar los dos modos del proyecto y cuando conviene cada uno.

## Modos disponibles

- Constructivista: enfoque interpretativo, flexible y sensible al contexto.
- Post-positivista: enfoque mas confirmatorio, orientado a contrastar evidencia.

## Referentes teoricos

- Constructivista: Charmaz.
- Post-positivista: Glaser.

## Guia de Seleccion

- Guía de Selección: resumen rapido para decidir modo.
- Cuando elegir Constructivista: exploracion temprana, categorias emergentes y foco en significado.
- Cuándo elegir Constructivista: cuando prima la interpretacion contextual.
- Cuando elegir Post-positivista: hipotesis mas definidas, comparabilidad y validacion sistematica.

## Comportamiento de lock

Una vez iniciado el flujo principal, el modo queda bloqueado (lock) para mantener coherencia metodologica.
Si se requiere cambiarlo, debe hacerse con un cambio controlado y documentado.
