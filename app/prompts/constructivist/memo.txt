ETAPA: MEMO REFLEXIVO (Enfoque Constructivista)

ENTRADA ESPERADA:
- memo_type: "codigo" | "teorico" | "reflexivo" | "metodologico"
- focus: string (código, categoría o tema del memo)
- codes_related: lista de {code_id: int, label: string} (opcional)
- evidence: lista de {evidence_id: int, text: string} (opcional)
- decisions_made: lista de strings (decisiones tomadas en la sesión, opcional)
- uncertainties: lista de strings (dudas o sesgos identificados, opcional)

OBJETIVO:
Documentar el proceso analítico con transparencia sobre el posicionamiento
del investigador y las decisiones metodológicas.

TIPOS DE MEMO:
1. MEMO DE CÓDIGO: Explora significados de un código específico
2. MEMO TEÓRICO: Conecta códigos/categorías en proposiciones
3. MEMO REFLEXIVO: Examina tu rol como investigador
4. MEMO METODOLÓGICO: Documenta decisiones de proceso

ESTRUCTURA DEL MEMO REFLEXIVO:
1. Posicionamiento: ¿Desde dónde analizo?
2. Decisiones: ¿Qué elegí enfatizar y qué dejé fuera?
3. Tensiones: ¿Qué contradicciones encuentro?
4. Preguntas: ¿Qué necesito explorar más?

INSTRUCCIONES:
1. Identifica el tipo de memo apropiado
2. Escribe en primera persona cuando sea reflexivo
3. Vincula a evidencia específica cuando aplique
4. Registra fecha y contexto

EJEMPLO MEMO REFLEXIVO:
{
  "tipo": "reflexivo",
  "fecha": "2026-01-23",
  "titulo": "Tensión entre mi lectura y las voces de los participantes",
  "contenido": "Al codificar las entrevistas sobre cambio demográfico, noto que mi propia experiencia como residente urbano puede estar influyendo en cómo interpreto la 'nostalgia'. Los participantes mayores usan este registro, pero ¿estoy proyectando mi propia ambivalencia hacia el cambio?",
  "preguntas_emergentes": [
    "¿Cómo hablan los participantes más jóvenes del mismo fenómeno?",
    "¿La nostalgia es un código in-vivo o una interpretación mía?"
  ],
  "accion_siguiente": "Revisar fragmentos de participantes < 40 años buscando variaciones"
}
